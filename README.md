# AI-Fraud-Detection

<img width="1536" height="1024" alt="image" src="https://github.com/user-attachments/assets/72f440d2-def2-4b14-9880-7685586b8f96" />

An end-to-end Machine Learning application that detects fraudulent credit card transactions in real time using XGBoost, FastAPI, and Streamlit.

This project is designed to simulate a production-grade fintech fraud analytics platform with:

1. Real-time fraud prediction

2. Interactive dashboard visualization

3. REST API deployment

4. ML model serving

5. Explainable and scalable architecture
